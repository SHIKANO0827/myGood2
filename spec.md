# Valorex Instagram スライド生成ツール 仕様書

## 概要

Valorex（本革スマホケース・財布・革小物ブランド）のInstagramリール・ストーリー用スライドを自動生成するツール。

---

## エージェント仕様

| 項目 | 内容 |
|------|------|
| 入力 | テーマテキスト（例：「本革の魅力」「ギフトにおすすめ」） |
| AI生成 | Claude APIで日英混在の文章を自動生成 |
| 出力 | 5枚のHTMLスライド（ブラウザ表示） |
| 保存 | 手動スクリーンショット → `pin/` フォルダへ |

---

## スライド構成（5枚）

| 枚数 | 役割 | 内容 |
|------|------|------|
| 1枚目 | 表紙 | タイトル・テーマ・ブランド名 |
| 2枚目 | 問題提起・共感 | ターゲットの悩みや共感ポイント |
| 3枚目 | 本文① | テーマの詳細・価値訴求 |
| 4枚目 | 本文② | 具体的なポイント・特徴 |
| 5枚目 | まとめ・CTA | フォロー促進・購買誘導 |

---

## デザイン仕様

| 項目 | 内容 |
|------|------|
| サイズ | 1080×1920px（9:16 縦長） |
| ブランドカラー | 淡いクリームピンク |
| 雰囲気 | 高級感×親しみやすさ（Bonaventura / Delvaux調） |
| フォント | セリフ体（Cormorant Garant など欧州ラグジュアリー系） |
| 言語 | 日英混在 |
| ロゴ | 各スライドに表示（`fe7b576e4455358a0be7c2ed3465326f.avif`） |
| アカウント名 | `@valorex_mgp`（全スライドに表示） |
| ターゲット層 | 30歳以上の男女 |

---

## ファイル構成

```
myGood2/
├── spec.md                         # 本仕様書
├── index.html                      # テーマ入力フォーム
├── slides.html                     # 生成されたスライド表示
├── generate.py                     # Claude API呼び出し・HTML生成
├── fe7b576e4455358a0be7c2ed3465326f.avif  # ロゴ画像
└── pin/                            # スクショ保存先
```

---

## 動作フロー

```
1. index.html をブラウザで開く
2. テーマテキストを入力して送信
3. generate.py が Claude API を呼び出してスライドコンテンツを生成
4. slides.html に5枚のスライドを表示
5. ブラウザで各スライドをスクリーンショット → pin/ に保存
```

## スライドデザイン詳細

### カラーパレット

| 用途 | カラー |
|------|--------|
| 背景 | クリームピンク（#FDF0EC など） |
| メインテキスト | ダークブラウン（#3D2B1F など） |
| アクセント | ゴールド（#C9A96E など） |
| サブテキスト | グレージュ（#8C7B74 など） |

### フォント

| 用途 | フォント |
|------|--------|
| ブランド名・見出し | Cormorant Garant（セリフ体） |
| 英語キャッチ | Playfair Display または EB Garamond |
| 日本語本文 | Noto Serif JP |

### 各スライド共通要素

- 左上または右上：ロゴ画像
- 下部：`@valorex_mgp`
- スライド番号（例：01 / 05）
